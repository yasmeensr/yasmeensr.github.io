Description

  • This dataset was sourced from the Kaggle website and was created by Tayyar Hussain. It was last updated 3 years ago. This dataset contains information about the best-selling game consoles of all time. The data includes the console name, manufacturer, release year, total units sold, and some other information. The dataset covers a wide range of consoles, from the earliest releases to the current generation of consoles.

Insights and Limitations

  • Through this data, I was able to see a lot of information I didn’t know about different types of consoles, the company, and how well they sell. Prior to this, I thought Microsoft would have had more sales, but compared to Sony and Nintendo, they don’t have nearly as many. I also was able to see what other brands sell gaming consoles that I didn’t know did like Nokia and Phillips. I was also able to see that home and handheld consoles were the most common type of console purchased. A limitation of this data is that it was last updated 3 years ago. I think if it were up to date, the data would look different, but I still think Sony, Nintendo, and Microsoft would be the top three.
