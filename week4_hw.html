Week 4 Homework: Wrangling with Pandas Using Heart Disease Data

Description: This dataset was sourced from the Kaggle website and was created by Huzaifa Ansari. It was last updated 2 months ago. This dataset contains real-world clinical attributes 🩺 used to analyze and predict the presence or absence of heart disease ❤️‍🔥. Each row represents one patient 👤, and each column represents a medical measurement or diagnostic indicator 🧠. This data required little cleaning, so I did more filtering to make up for it. I know when cleaning, though, you should get rid of duplicates, handle missing values, and fix data types, but this data set didn’t really require that.

4) Reshaping the Data

#Using Pivot to see average age by sex and heart disease
reshaped = heart_data_clean.pivot_table(
    index='Sex', 
    columns='Heart Disease', 
    values='Age', 
    aggfunc='count'
)

#A Plot stacked barplot
reshaped.plot(
    kind="bar",
)

plt.title("Heart Disease By Sex")
plt.xlabel('Sex (0=Female, 1=Male)')
plt.ylabel('Count')
plt.legend(title='Heart Disease')
plt.xticks(rotation=0)
plt.show()
